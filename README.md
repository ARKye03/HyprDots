## Usage

  Install stow, dependencies and clone the repo.
  
  ```sh
  git clone https://github.com/ARKye03/HyprDots.git
  cd HyprDots
  stow . --dotfiles
  ```

  Have in mind that if stow finds a file that already exists, it will not overwrite it.
